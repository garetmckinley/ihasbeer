Vim contains 3 primary modes:

These keys move the cursor one position in their respective direction. It‚Äôs pretty bad practice to just repeatedly tap away on these keys to get where you wanna go though. There‚Äôs much faster ways to navigate, like these (keys are case-sensitive in vim, all uppercase letters represent shift+letter):

  • 0 = move to the beginning of the line
  • $ = move to the end of the line
  • w = move to the beginning of the next word
  • b = move to the beginning of the previous word

You can also add a number before any movement command to repeat it x amount of times. i.e. 2w moves the cursor forward 2 words.

Insert Mode

insert mode can be entered by pressing i on the keyboard. When inside insert mode, every keystroke will be immediately entered into the editor. To exit insert mode and return to normal mode, press the esc key. If you are using the new Macbook Pro with a touch bar, may God be with you.

Visual Mode

visual mode is activated when selecting content. There are multiple ways to enter visual mode, the most common being v (begin selection at cursor) and shift+v (select current line). When in visual mode, all the movement commands available in normal mode will be available for use. Every movement will expand/reduce the selection depending on the direction you go.

It‚Äôs pretty much the same as holding down the shift key and navigating around in any other text editor with the arrow keys, except you have the added benefit of being able to use the complex movement commands to quickly select exactly what you want to.
